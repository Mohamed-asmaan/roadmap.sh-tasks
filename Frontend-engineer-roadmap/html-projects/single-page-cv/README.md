# Single Page CV

This project is part of the roadmap.sh Frontend Engineer roadmap.

## Objective
Build a single-page CV using only HTML, focusing on:
- Semantic HTML
- Proper document structure
- Accessibility-friendly markup

## Constraints
- HTML only
- No CSS
- No JavaScript

## Project link
https://roadmap.sh/projects/single-page-cv

## How to run
Open `index.html` in a browser.
